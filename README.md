
# vimrc

My vim configure files.

## Usage

### Windows

```bash
# cmd
git clone https://github.com/pexcn/vimrc.git %USERPROFILE%\vimfiles
cd %USERPROFILE%\vimfiles
git submodule init
git submodule update

# cygwin
git clone https://github.com/pexcn/vimrc.git $USERPROFILE/vimfiles
cd $USERPROFILE/vimfiles
git submodule init
git submodule update
```

### Unix

```bash
git clone https://github.com/pexcn/vimrc.git $HOME/.vim
cd $HOME/.vim
git submodule init
git submodule update
```
